# Azure VM Automated Deployment

This project demonstrates how to automate the deployment of a Virtual Machine in Microsoft Azure using Azure CLI.

---

## Technologies Used

* Microsoft Azure
* Azure CLI
* Bash Scripting
* Linux

---

## What This Script Does

1. Creates a Resource Group
2. Deploys an Ubuntu Virtual Machine
3. Generates SSH keys
4. Opens port 22 for SSH access

---

## How to Run

1. Install Azure CLI
2. Login to Azure

az login

3. Run the script

chmod +x deploy-vm.sh
./deploy-vm.sh
